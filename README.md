# <img src="[https://wearemobi.com/icon-light.svg](https://wearemobi.com/icon-light.svg)" width="24" height="24" valign="middle"\> mobi-signature

Professional digital identity engine for high-performance networking. Built with Next.js 16, Tailwind 4, and M.O.B.I.™ architecture.

## 🚀 The Single-Pilot Edition (v1.8)

This version is optimized for rapid deployment and professional networking. One repository, one identity, zero friction.

### 🛠️ Quick Start

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/wearemobi/mobi-signature.git
    cd mobi-signature
    ```

2.  **Personalize your identity:**
    Edit `src/data/profile.json` with your professional details.

3.  **Install and Build:**

    ```bash
    npm install
    npm run build
    ```

4.  **Deploy:**
    Connect your repository to **Cloudflare Pages** and set the build output directory to `out`.

> **Pro Tip (v1.8):** For client-specific pilots, use the `signature-[name]` naming convention and sync with this repo as your `upstream` for future core updates.

## 🎨 Features

- **Print Engine v1.8:** High-contrast optimization for monochrome laser printers (HP LaserJet M111w).
- **M.O.B.I.™ Branding:** Integrated networking footer with recursive traffic links.
- **Performance Optimized:** Zero-cascading renders and sanitized codebase for \<60s build times.
- **Static Export:** Blazing fast performance via Next.js `output: 'export'`.
