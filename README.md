# <img src="https://wearemobi.com/icon-light.svg" width="24" height="24" valign="middle"> mobi-signature
Professional digital identity engine for high-performance networking. Built with Next.js 16, Tailwind 4, and M.O.B.I.™ architecture.

## 🚀 The Single-Pilot Edition

This version is designed for rapid, individual deployment. One repository, one identity, zero friction.

### 🛠️ Quick Start

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/wearemobi/mobi-signature.git](https://github.com/wearemobi/mobi-signature.git)
   cd mobi-signature
   ```

2. **Personalize your identity:**
   Edit `src/data/profile.json` with your professional details.

3. **Install and Build:**
   ```bash
   npm install
   npm run build
   ```
   > Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
4. **Deploy (Optional):**
   Connect your repository to **Cloudflare Pages** and set the build output directory to `out`.

- **SVG Branding:** Ultra-sharp M.O.B.I. visual identity.
- **Static Export:** Blazing fast performance and worldwide availability.
